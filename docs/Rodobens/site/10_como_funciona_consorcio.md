# Como Funciona o Consórcio

## Visão Geral do Processo

O consórcio funciona através de 4 etapas principais:

1. **SIMULE** - Calculadora online
2. **COMECE** - Primeira parcela e contrato
3. **ACOMPANHE** - Participação em assembleias
4. **CONTEMPLAÇÃO** - Receba seu bem

Tempo total: Variável conforme sorteio ou lance

## Etapa 1: SIMULE

### O Que É
Na etapa de simulação, você descobre quanto vai pagar mensalmente pelo seu bem desejado.

### Como Fazer
```
1. Acesse www.rodobens.com.br
2. Selecione tipo de consórcio
   (Carros, Imóveis, Motos, Caminhões, Serviços)
3. Digite especificações:
   - Modelo/tipo específico
   - Valor aproximado do bem
   - Prazo desejado (meses)
4. Clique em "Simular"
5. Visualize valor da parcela
```

### Exemplo de Simulação
```
CONSÓRCIO DE CARROS
├─ Modelo: Toyota Corolla
├─ Prazo: 60 meses
└─ Resultado: R$ 847,50 por mês

CONSÓRCIO DE IMÓVEL
├─ Valor: R$ 300 mil
├─ Prazo: 120 meses
└─ Resultado: R$ 2.500 por mês

CONSÓRCIO PONTUAL (Carro)
├─ Modelo: Chevrolet Onix
├─ Prazo: 6 meses
└─ Resultado: R$ 700 por mês
```

### Informações Fornecidas
Na simulação você visualiza:
- ✅ Valor da parcela
- ✅ Número de parcelas
- ✅ Valor total
- ✅ Taxa de administração
- ✅ Fundo de reserva
- ✅ Despesa com assembleia
- ✅ Documentação necessária

### É Obrigatório Contratar?
**Não.** A simulação é apenas para conhecer valores. Você pode simular sem compromisso.

## Etapa 2: COMECE

### O Que É
Nesta etapa, você formaliza a contratação do consórcio com a Rodobens.

### Passo-a-Passo

#### 2.1 - Cadastro Online (10-15 minutos)
```
Acesse www.rodobens.com.br e preencha:

DADOS PESSOAIS:
├─ Nome completo
├─ Data de nascimento
├─ CPF
├─ E-mail
├─ Telefone
└─ Sexo

DADOS DE ENDEREÇO:
├─ Rua e número
├─ Complemento
├─ Bairro
├─ Cidade
├─ Estado
├─ CEP
└─ Tempo de residência

DADOS PROFISSIONAIS:
├─ Profissão
├─ Empresa onde trabalha
├─ Tempo de empresa
├─ Renda mensal
├─ Tipo de contrato (CLT, autônomo, etc)
└─ Dados do empregador
```

#### 2.2 - Envio de Documentação (1-2 dias)
```
Documentos necessários:

ESSENCIAIS:
├─ RG (foto nítida, frente e verso)
├─ CPF (foto nítida)
├─ Comprovante de endereço
│  (conta de água, luz, gás dos últimos 3 meses)
└─ Comprovante de renda
   ├─ Se CLT: últimos 3 contracheques
   ├─ Se autônomo: últimos 3 recibos (RPA)
   ├─ Se aposentado: contracheque + comprovação INSS
   └─ Se PJ: últimas 2 declarações de IR

DADOS BANCÁRIOS:
├─ Banco (nome e número)
├─ Agência
├─ Conta corrente/poupança
├─ CPF/CNPJ do titular
└─ Tipo de conta

ADICIONAIS (conforme tipo):
├─ Se imóvel: certidão de propriedade
├─ Se PJ: contrato social, CNPJ
└─ Se servidor público: matrícula
```

#### 2.3 - Validação (24-48 horas)
```
Rodobens verifica:
✓ Documentação completa
✓ Dados consistentes
✓ Consulta SCPC/SPC (crédito)
✓ Validação de endereço
✓ Renda compatível
✓ Registro Banco Central
✓ Antecedentes (segurança)

Status: Aceito ou Rejeito
Motivo: Enviado por e-mail
Recurso: Pode contestar ou reagendar
```

#### 2.4 - Aprovação e Contrato (1-2 dias)
```
Se aprovado, você recebe:

1. E-MAIL COM:
   ├─ Contrato em PDF
   ├─ Dados do consórcio
   ├─ Número da cota
   ├─ Condições especiais
   ├─ Instruções de assinatura
   └─ Data de vencimento primeira parcela

2. CONTRATO DIGITAL:
   ├─ Assinatura eletrônica
   ├─ via email ou plataforma
   ├─ Uso de código de segurança
   ├─ Geração automática
   └─ Arquivo armazenado

3. CONFIRMAÇÃO:
   └─ Comprovante de contrato assinado
```

#### 2.5 - Primeira Parcela (Imediato após assinatura)
```
Após assinar:

1. BOLETO GERADO:
   ├─ Enviado por e-mail
   ├─ Disponível na plataforma
   ├─ Código de barras para pagamento
   ├─ Data de vencimento: 30 dias
   └─ Valor da primeira parcela

2. DÉBITO AUTOMÁTICO (se optado):
   ├─ Configurado na conta bancária
   ├─ Débito no dia combinado
   ├─ Confirmação por SMS/e-mail
   ├─ Comprovante disponível
   └─ Sem taxa extra

3. PAGAMENTO:
   ├─ Banco (caixa)
   ├─ Lotérica
   ├─ Internet banking
   ├─ Aplicativo Rodobens
   └─ Débito automático

4. CONFIRMAÇÃO:
   └─ Após pagamento, começa a contar
```

### O Que É
Período em que você participa do consórcio, paga parcelas e participa das assembleias mensais.

```
✓ Extratos completos
✓ Documentos
✓ Comunicados
```

#### Aplicativo Mobile
```
Disponível em iOS e Android

Funcionalidades:
✓ Acompanhamento em tempo real

✓ Geração de boletos
✓ Comprovante de pagamento

✓ Consulta de saldo
✓ Histórico de operações
```

### Participação em Assembleias

#### O Que é Assembleia
Reunião mensal onde se sorteiam as pessoas que vão receber seu bem.

#### Frequência
- **Mensal**: Uma vez por mês
- **Data**: Geralmente primeira semana
- **Horário**: 14:00 ou 19:00 (conforme escolha)
- **Local**: Presencial ou online

#### Como Funciona

```
ASSEMBLEIA PRESENCIAL:
├─ Local: Escritório Rodobens (sua região)
├─ Horário: 14:00 ou 19:00
├─ Duração: 30-60 minutos
├─ Presença: Obrigatória para votar/lançar
├─ Documentação: Levar RG/CPF
└─ Resultado: Ata assinada entregue

ASSEMBLEIA ONLINE:
├─ Plataforma: Zoom/Teams
├─ Acesso: Link enviado por e-mail
├─ Horário: 19:00 (geralmente)
├─ Duração: 30-45 minutos
├─ Participação: Chat/voto eletrônico
└─ Resultado: Ata digital enviada
```

#### Etapas da Assembleia

```
1. ABERTURA (5 minutos)
   ├─ Saudação aos participantes
   ├─ Apresentação da pauta
   ├─ Validação de quórum
   └─ Leitura de comunicados

2. SORTEIO (10 minutos)
   ├─ Extração de números
   ├─ Confirmação de pagamentos
   ├─ Seleção do contemplado
   └─ Verificação de adimplência

3. LANCES (15-20 minutos)
   ├─ Abertura para lances
   ├─ Participantes oferecem valores
   ├─ Lances registrados
   ├─ Lance maior vence
   └─ Contabilização

4. RESULTADO (5 minutos)
   ├─ Proclamação do contemplado
   ├─ Lance ganhador (se houver)
   ├─ Dados para contato
   └─ Próximas datas
```

### O que é Lance (Oferta)

#### Conceito
Lance é uma oferta de valor adicional que você faz para ser contemplado antecipadamente.

#### Como Funciona
```
Exemplo: Consórcio de R$ 10 mil

1. SITUAÇÃO NORMAL:
   ├─ Você aguarda ser sorteado
   ├─ Semana 1: não sorteado
   ├─ Semana 2: não sorteado
   ├─ Semana 3: não sorteado
   └─ Espera indefinidamente

2. COM LANCE:
   ├─ Você oferece um valor extra
   ├─ Exemplo: R$ 200 de lance
   ├─ Semana 1: Oferece lance
   ├─ Naquela semana: É contemplado
   └─ Ganhou a concorrência
```

#### Vantagem do Lance
- ✅ Antecipa recebimento do bem
- ✅ Coloca você à frente de outros
- ✅ Valor reflete em parcelas futuras
- ✅ Decisão pessoal e estratégica

#### Valor do Lance
```
Não há limite máximo
Você decide o valor do seu lance

Exemplos:
├─ Lance conservador: R$ 100-200
├─ Lance agressivo: R$ 500-1.000
├─ Lance máximo: Quanto você conseguir pagar
└─ Ajusta-se à sua situação
```

### Pagamento de Parcelas

#### Formas de Pagamento
```
1. BOLETO TRADICIONAL:
   ├─ Enviado por e-mail
   ├─ Pagar em banco/lotérica
   ├─ Prazo até vencimento
   └─ Comprovante gerado

2. DÉBITO AUTOMÁTICO:
   ├─ Configurado 1x
   ├─ Débito automático mensal
   ├─ Mesma data do mês
   ├─ Sem taxa adicional
   └─ Mais comodidade

3. TRANSFERÊNCIA BANCÁRIA:
   ├─ Dados fornecidos
   ├─ TED/DOC possível
   ├─ Confirmação automática
   └─ Comprovante gerado

4. APLICATIVO RODOBENS:
   ├─ Boleto gerado na hora
   ├─ Pagamento direto
   ├─ Confirmação instantânea
   └─ Arquivo salvo
```

#### Multa e Atraso
```
Se atrasar:
├─ Multa moratória: ~2%
├─ Juros: ~1% ao mês
├─ Juros compostos após 30 dias
└─ Pode afetar contemplação

Recomendação:
├─ Pagar no prazo
├─ Configurar débito automático
├─ Manter contato com Rodobens
└─ Avisar antecipadamente se problema
```

### Posição na Fila

Você tem duas formas de avançar:

```
1. FILA NATURAL:
   ├─ Avança a cada mês
   ├─ Automático
   ├─ Sem necessidade de ação
   ├─ Quantidade fixa por mês
   └─ Tempo: varia (meses/anos)

2. SORTEIO:
   ├─ Chance mensal
   ├─ Aleatório
   ├─ Pode vencer rápido
   ├─ Ou demorar
   └─ Sorte do acaso

3. LANCE (Oferta):
   ├─ Você paga extra
   ├─ Vence na concorrência
   ├─ Garantido
   ├─ Quando desejar
   └─ Acelera processo
```

## Etapa 4: CONTEMPLAÇÃO

### O Que Significa
Contemplação é quando você é escolhido (sorteio ou lance) e pode receber seu bem.

### Como Recebe Seu Bem

#### Processo
```
1. CHAMADA PELA RODOBENS:
   ├─ Ligação telefônica
   ├─ E-mail confirmando
   ├─ SMS com dados
   ├─ Parabéns! Você foi contemplado
   └─ Prazo de reclamação: 30 dias

2. DOCUMENTAÇÃO SOLICITADA:
   ├─ RG original (se pessoa física)
   ├─ CPF original
   ├─ Comprovante de renda recente
   ├─ Comprovante de endereço
   ├─ Dados bancários (para CRLV)
   └─ Carteira de motorista (carros/motos)

3. AGENDAMENTO:
   ├─ Marca data de entrega
   ├─ Local: Concessionária parceira
   ├─ Horário: Combinado com você
   ├─ Documentação necessária
   └─ Estimativa: 5-10 dias úteis

4. ENTREGA DO BEM:
   ├─ Bem entregue em seu nome
   ├─ Documentação completa
   ├─ Para carro:
   │  ├─ DUT (Documento Único de Trânsito)
   │  ├─ CRLV (Certificado de Registro)
   │  ├─ Manual do proprietário
   │  ├─ Documentação de garantia
   │  └─ Chaves (todas)
   ├─ Para imóvel:
   │  ├─ Escritura pública
   │  ├─ Registro no cartório
   │  ├─ Matrícula atualizada
   │  ├─ Certidão de ônus
   │  └─ Recibos de taxas
   └─ Fotos de entrega
```

### Despesas de Contemplação

```
PARA CARROS/MOTOS:
├─ IPVA (anual): Você paga
├─ Seguro obrigatório: Incluído
├─ Emplacamento: ~R$ 200-500
├─ Transferência: ~R$ 200-300
├─ Documentação: Rodobens cobre
└─ Total adicional: ~R$ 500-800

PARA IMÓVEL:
├─ Escritura pública: Rodobens cobre
├─ Cartório: Você paga (~R$ 500-2000)
├─ ITBI (imposto): Você paga (~2-3%)
├─ Registro imóvel: ~R$ 200-500
├─ Documentação: Rodobens cobre
└─ Total: Varia conforme estado

PARA SERVIÇOS:
├─ Contratação do prestador: Você faz
├─ Valores conforme serviço
└─ Rodobens apenas fornece crédito
```

### Após Contemplação

```
Você agora tem:

✓ Bem em seu nome
✓ Documentação completa
✓ Posse legal
✓ Registro atualizado
✓ Direito de revenda
✓ Herança para família
✓ Garantia do fabricante (carros/motos)

Obrigações:
├─ Manter pagamentos pontuais
├─ Se houver parcelas restantes
├─ Pagar impostos (IPVA, ITBI, etc)
├─ Manter seguro em dia
├─ Cumprir regulamentações
└─ Manter documentação guardada
```

## Cronograma Estimado

### Consórcio de Carros (60 meses)

```
Tempo com Sorteio:
├─ Sem lance: 24-48 meses (média)
└─ Pode variar muito

Tempo com Lance (Acelerado):
├─ 1º mês: oferece lance
├─ 1ª assembleia: pode vencer
├─ 2-4 semanas: entrega
└─ Tempo total: 6-8 semanas
```

### Consórcio Pontual (6 meses)

```
Cronograma Fixo:
├─ Mês 1: Paga 1ª parcela
├─ Mês 2: Paga 2ª parcela
├─ Mês 3: Paga 3ª parcela
├─ Mês 4: Paga 4ª parcela
├─ Mês 5: Paga 5ª parcela
├─ Mês 6: Paga 6ª parcela
├─ Semanas 7-8: Entrega
└─ Tempo total: 6-8 meses
```

### Consórcio de Imóvel (240 meses)

```
Com Sorteio Rápido:
├─ Tempo: 24-60 meses

Com Lance Agressivo:
├─ Quando desejar: oferece lance
├─ Próxima assembleia: contemplação
└─ 2-4 semanas: Entrega

FGTS + Lance:
├─ Reduz saldo devedor
├─ Aceleraa contemplação
├─ Tempo: 6-12 meses
└─ Recomendado
```
