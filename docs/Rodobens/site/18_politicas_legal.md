```
└─ Gênero

PROFISSIONAIS:
├─ Profissão
├─ Empresa
├─ Cargo
├─ Tempo de empresa
├─ Renda mensal
└─ Tipo de contrato

FINANCEIROS:
├─ Conta bancária
├─ Agência

├─ Histórico de crédito
├─ Consultas SPC/SCPC
└─ Outros créditos

COMPORTAMENTAIS:
├─ Histórico consórcio
├─ Frequência de consultas
├─ Padrão de pagamento
├─ Produtos contratados
├─ Reclamações
└─ Feedback
```

#### Uso dos Dados
```
FINALIDADES:

├─ Comunicações comerciais
├─ Melhorias de serviço
├─ Conformidade regulatória
├─ Segurança e fraude
└─ Pesquisa e análise
```

#### Compartilhamento
```
COMPARTILHA COM:
├─ Seguradoras parceiras
├─ Órgãos reguladores (Banco Central, SUSEP)
├─ Serviços de proteção (SPC, SCPC)
├─ Prestadores de serviço (avaliadores, etc)
├─ Autoridades (quando exigido)
└─ Nunca: Redes sociais, terceiros não autorizados

NÃO COMPARTILHA:
├─ Dados com empresas de marketing
├─ Senha ou código de segurança
├─ Dados a concorrentes
├─ Sem consentimento prévio
└─ Fora dos casos legais
```


## Conformidade Regulatória

### Banco Central do Brasil

**Supervisão**: Consórcio é regulado pelo Banco Central

```
CONFORMIDADE:
├─ Registro junto Banco Central
├─ Cumprimento de limites de crédito
├─ Reservas obrigatórias mantidas
├─ Auditoria regular
├─ Relatórios de conformidade
├─ Inspeções periódicas
└─ Regime de solvência
```

### SUSEP (Superintendência de Seguros Privados)

**Supervisão**: Seguros são regulados por SUSEP

```
CONFORMIDADE:
├─ Autorização para vender seguros
├─ Seguradoras parceiras certificadas
├─ Produtos aprovados por SUSEP
├─ Transparência em coberturas
├─ Limite de responsabilidade
├─ Fundo de garantia
└─ Reclamações em SUSEP
```

### MESC (Sindicato das Administradoras)

**Filiação**: Rodobens é membro do MESC

```
BENEFÍCIOS:
├─ Representação industrial
├─ Normatização técnica
├─ Formação profissional
├─ Defesa de interesses
├─ Qualidade e padronização
└─ Ética no setor
```

### LGPD (Lei Geral de Proteção de Dados)

**Compliance**: Rodobens segue rigorosamente LGPD

```
PRINCÍPIOS:
├─ Finalidade específica
├─ Transparência
├─ Segurança dos dados
├─ Consentimento
├─ Responsabilidade
└─ Respeito à privacidade

IMPLEMENTAÇÃO:
├─ Política de privacidade
├─ Criptografia de dados
├─ Acesso controlado
├─ Auditoria de dados
├─ Equipe DPO (Data Protection Officer)
└─ Treinamento de funcionários
```
