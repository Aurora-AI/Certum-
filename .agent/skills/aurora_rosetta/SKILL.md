# Aurora Rosetta (Sovereign Compass)

> "Nos não construímos sites. Nós damos à luz Entidades. Tudo dentro do Mad Lab Aurora deve ser disruptivo, ter movimento, 3D, ter profundidade e principalmente que quebre as leis da física."

## O Manifesto
Este documento define a alma do ecossistema Mad Lab Aurora. Não é apenas um guia de estilo; é um conjunto de leis físicas e filosóficas que regem toda criação sob a bandeira Aurora.

## Princípios Fundamentais (The 5 Laws)

### 1. PROFUNDIDADE OBRIGATÓRIA (Z-Axis Mandate)
Todo efeito opera em Z ou simula profundidade. O "Flat Design" é estritamente proibido. A tela não é uma superfície; é uma janela para um espaço volumétrico.

### 2. CALL & RESPONSE (Blues Philosophy)
Toda ação do usuário (scroll, hover, click) recebe uma resposta espacial proporcional.
- **Micro**: Cursor magnetismo, tilt de cards.
- **Macro**: Navegação de página, distorção de velocidade.
Nada é inerte. Se o usuário toca, o digital responde.

### 3. TRUSTWARE GATE
Se um efeito puder ser removido sem alterar a função cognitiva ou a narrativa espacial, ele é inválido (apenas "enfeite"). 
- **Teste**: "Se eu tirar isso, o site vira uma planilha?" Se sim, o efeito é **Trustware** (Essencial).

### 4. FALLBACK MOBILE
Todo efeito deve declarar explicitamente seu comportamento degradado para dispositivos móveis. A experiência não pode quebrar, deve se adaptar mantendo a alma (mesmo que simplificada).

### 5. LEI DA PROPORCIONALIDADE
Intensidade da resposta <= Intensidade da pergunta.
- Um **hover** suave não pode causar uma explosão de partículas.
- Um **click** decisivo deve ter uma transição massiva.

## Trustware Gates Definition
O conceito de **Trustware Gate** é o filtro final de qualidade.
- **VALIDO**: O efeito carrega função cognitiva (ex: hierarquia visual via profundidade, feedback via física).
- **INVALIDO**: O efeito é puramente decorativo e desconectado da ação do usuário.
